# java-ladder

사다리 타기 미션 저장소

## 기능 목록

- [x] 사용자 입력(사람 이름은 쉼표(,)를 기준으로 구분)
- [x] 사다리 높이 입력
- [x] 랜덤 값(0,1) 생성 기능
- [x] 사다리 라인 생성
    - [x] 이전 좌표와 겹치는지 판단
- [x] 입력받은 이름 글자 수 최대의 따라 사다리 폭 결정
- [x] 플레이어 이름의 길이의 따라 사다리 폭 변화 후 출력
    - [x] 플레이어 중 이름의 최대길이 구하기 기능
    - [x] 사다리 좌표값 enum 생성
- [x] player마다 결과 저장
    - [x] 출발 위치의 따른 결과 저장
        - [x] 오른쪽,왼쪽,정지할지 판다.

## 예외 처리

- [x] 사용자 이름 글자 수가 0이하 이거나 5를 초과했을때.
- [x] 사용자 이름을 입력하지 않았을때.
- [x] 사용자 이름이 중복일때.
- [x] 입력된 사다리 높이가 숫자가 아닐때.
- [x] 입력된 사다리 높이가 1이상인 정수가 아닐때.
- [x] 상품을 입력하지 않았을때.
- [x] 상품의 개수가 1개 미만 5개 초과일때.
